# Visualization SS26

Repository containing the solutions for the exercises of the "Visualization" course (HTW Berlin Applied Computer Science, Master).

## Included Exercise Solutions

### Exercise 1: Perceived Area Experiment

Design and implement a program to measure the perception exponent `x` for circles and squares, run a measurement series with one or more participants, and visualize the resulting data in a suitable way.

Folder: [`excercise_1`](https://github.com/Sylver2k/visualization-ss26/tree/main/excercise_1)

### Exercise 2: Preattentive Perception Test

Design and implement a program to test whether differente visual object properties can be perceived pre-attentively.

Folder: [`excercise_2`](https://github.com/Sylver2k/visualization-ss26/tree/main/excercise_2)

### Exercise 3: Car Dataset Visualization Concept

Develop a concept and game plan for an interactive glyph-based visualization of a car dataset, including suitable visual encodings, layout, and interactions.

Folder: [`excercise_3`](https://github.com/Sylver2k/visualization-ss26/tree/main/excercise_3)

### Exercise 4: Car Dataset Visualization

Implement the planned interactive glyph-based visualization of the car dataset, allowing users to compare attributes, filter missing values, and inspect the details of individual cars.

Folder: [`excercise_4`](https://github.com/Sylver2k/visualization-ss26/tree/main/excercise_4)
